GetInventoryItems

Returns all slots with items in a inventory.

exports.ox_inventory:GetInventoryItems(inv, owner)
  • inv: number or table
  • owner?: string or boolean

Example:

local playerItems = exports.ox_inventory:GetInventoryItems(source)
